JOB TITLE: Full Stack Developer
COMPANY: TechStartup Co.
LOCATION: Remote (US-based)
JOB TYPE: Full-time
SALARY: $120,000 - $160,000

ABOUT TECHSTARTUP CO.
TechStartup Co. is a fast-growing SaaS company building the future of team collaboration. 
Our platform helps remote teams stay connected and productive through innovative features 
like real-time document editing, video conferencing, and project management tools. 
We're a team of 50+ passionate builders working to change how teams work together.

THE OPPORTUNITY
We're looking for a talented Full Stack Developer to join our Product Engineering team. 
You'll be building features that millions of users interact with daily, working across 
our entire tech stack from React frontends to Node.js microservices. This is a great 
opportunity for someone who loves shipping features fast while maintaining code quality.

WHAT YOU'LL DO
• Build and maintain web applications using React, TypeScript, and Node.js
• Design and implement RESTful and GraphQL APIs
• Create responsive, accessible user interfaces
• Write clean, maintainable, and well-tested code
• Participate in architecture discussions and technical design reviews
• Collaborate with designers, product managers, and other engineers
• Debug production issues and implement fixes
• Contribute to our engineering blog and documentation

REQUIREMENTS
• 3+ years of professional experience in full-stack web development
• Strong proficiency in JavaScript/TypeScript
• Experience with React (or similar framework: Vue, Angular)
• Experience with Node.js and Express (or similar: FastAPI, Django)
• Familiarity with both SQL (PostgreSQL, MySQL) and NoSQL (MongoDB, Redis) databases
• Understanding of RESTful API design principles
• Experience with version control (Git) and CI/CD pipelines
• Strong problem-solving and communication skills
• Comfortable working in an agile environment

NICE TO HAVE
• Experience with Docker and Kubernetes
• Knowledge of GraphQL
• Experience with AWS or GCP cloud services
• Familiarity with real-time technologies (WebSockets, Socket.io)
• Contributions to open source projects
• Experience with testing frameworks (Jest, Cypress, Playwright)
• Background in building collaborative or real-time applications

OUR TECH STACK
• Frontend: React, TypeScript, Next.js, Tailwind CSS
• Backend: Node.js, Express, GraphQL, PostgreSQL, Redis
